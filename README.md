# Привіт, я Barneli!  
### Спеціаліст із розробки ПЗ та управління базами даних

Я — розробник із фокусом на архітектурі баз даних та створенні десктопних додатків. Маю значний практичний досвід у проектуванні та оптимізації реляційних систем (SQL/MySQL) та понад 1.5 роки роботи з мовою C++. Спеціалізуюся на створенні комплексних рішень, що поєднують надійну серверу логіку та зручні інтерфейси користувача.

---

## Технічні навички
* **Бази даних:** MySQL, PostgreSQL, SQL Server, MS Access.
* **Мови програмування:** SQL (DML, DDL), C++ (VCL, Qt), Python, PHP.
* **Інструменти:** MySQL Workbench, phpMyAdmin, Qt Creator, C++ Builder.
* **Методології:** Нормалізація БД, оптимізація запитів, розробка ER-діаграм та GUI-додатків.

## Мої проєкти

### 1. [Гра «Морський бій» (Sea Battle)](https://github.com/Barneli/Sea-Battle-CPP-Builder)
Повноцінна десктопна гра «Морський бій» із коректною логікою боїв та системою рейтингу.
* **Мета:** Створити ігровий додаток із режимами гри проти бота та по локальній мережі.
* **Інструменти:** C++ Builder (VCL), Sockets для мультиплеєра.
* **Результати:** Функціональна гра з автоматизованим суперником та стабільним мережевим з'єднанням.

### 2. [Система БД «Магазин будівельних матеріалів»](https://github.com/Barneli/Database)
Реляційна система управління для магазину будівельних матеріалів.
* **Мета:** Автоматизація обліку товарів, клієнтів та персоналу за допомогою складних SQL-запитів.
* **Опис:** Включає дві інтегровані бази даних (`Sales` та `Inventory`) з повною нормалізацією таблиць.
* **Візуалізація:** Проєкт містить детальні EER-діаграми структури.

### 3.[Сервіс скорочення посилань (URL Shortener)](https://github.com/Barneli/URL-Shortener-App) 
Десктопна утиліта для скорочення довгих URL-адрес.
* **Інструменти:** C++, Qt Framework, SQL інтеграція для керування посиланнями.
* **Функціонал:** Створення коротких псевдонімів, їх збереження в БД та автоматичне перенаправлення.

### 4. [Колекція прикладних баз даних MS Access](https://github.com/Barneli/MS-Access-Projects)
Масштабна колекція з понад 20 прикладних баз даних для різних галузей бізнесу.
* **Ключові проєкти:** Аеропорт, Будівельна фірма, Магазин комп'ютерної техніки, Зоомагазин, Юридична фірма.
* **Технології:** Складна схема даних (Relationships), автоматизовані форми та звіти.

### 5. [Система управління блогом «New_Blog»](https://github.com/Barneli/New_Blog)
Система управління контентом для ведення особистого блогу.
* **Функціонал:** Додавання, редагування та перегляд постів з використанням SQL для постійного зберігання контенту.
